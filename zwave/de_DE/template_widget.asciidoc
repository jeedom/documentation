==== Visuelle Alternative

{nbsp} +

Wir müssen jetzt das Modul inkludieren, dazu müssen wir in das Menü gehen « Plugins », « Automatisierungsprotokoll » dann « Z-Wave ».

{nbsp} +

image::../images/plugin/inclusion1.jpg[width=556]

{nbsp} +

Einmal auf dieser Seite, finden Sie die "Inklusions Modus" Schaltfläche, klicken Sie einfach drauf um in den Inklusions Modus zu wechseln :

{nbsp} +

image::../images/plugin/bouton_inclusion.jpg[width=256]

{nbsp} +

Nach einem Klick auf « Inklusions Modus », erscheint eine Meldung, die Sie daran erinnert, dass Sie in diesem Modus sind.

{nbsp} +

image::../images/plugin/inclusion3.jpg[width=756]

{nbsp} +

Sobald die Inklusion läuft, wird eine Meldung angezeigt, das es notwendig ist zu warten, während Jeedom das Modul integriert.
Die Befehls Konfiguration erfolgt automatisch, sobald Jeedom das Modul erkannt hat.

{nbsp} +

image::../images/plugin/inclusion4.jpg[width=756]

{nbsp} +

[icon="../images/plugin/tip.png"]
[TIP]
Am besten ist es eine Exklusion vor der Inklusion zu machen. Die Exclusion ist nicht kompliziert durchzuführen,
klicken Sie einfach auf « Exklusions Modus » um das Modul zu entfernen, es erscheint eine Nachricht, die das Entfernen des Moduls bestätigt.

{nbsp} +
